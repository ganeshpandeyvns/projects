## Design Notes

### Colors
- Primary: Coral/Red (#E85D5D)
- Background: White/Light gray
- Text: Dark navy (#1A1A2E)
- Accent: Gold for highlights

### Typography
- Clean sans-serif (likely Inter or similar)
- Bold headings
- Clear hierarchy

### Key UI Patterns
- Card-based layouts
- Left sidebar navigation
- Data tables with badges
- Chart with timeframe toggles
- Form inputs with clear labels
